# Form Input Validation Requirements

Fields marked with 🔴 are **required**.

---

## 1. First Name 🔴

**Required:** Yes  

### Rules
- Must not be empty.
- Only alphabetic characters allowed (`A–Z`, `a–z`).
- Spaces allowed for compound names (e.g., `Mary Anne`).
- Must not contain numbers.
- Must not contain special characters or symbols (`@ # $ % !`, etc.).
- Length must be **2–50 characters**.

### Example Regex
^[A-Za-z]+(?: [A-Za-z]+)*$


---

## 2. Middle Name

**Required:** No  

### Rules
- Alphabetic characters only.
- Spaces allowed.
- Must not contain numbers.
- Must not contain special characters.
- Length must be **0–50 characters**.

### Example Regex
^[A-Za-z](?: [A-Za-z]+)$


---

## 3. Last Name 🔴

**Required:** Yes  

### Rules
- Must not be empty.
- Only alphabetic characters allowed.
- Spaces allowed for compound names (e.g., `De Leon`).
- Must not contain numbers.
- Must not contain special characters.
- Length must be **2–50 characters**.

### Example Regex
^[A-Za-z]+(?: [A-Za-z]+)*$


---

## 4. Suffix

**Required:** No  

### Rules
- Must match recognized suffix values.
- Allowed values:
  - `Jr`
  - `Sr`
  - `II`
  - `III`
  - `IV`
  - `V`
- No other numbers or special characters allowed.

### Example Regex
^(Jr|Sr|II|III|IV|V)?$


---

## 5. Address 🔴

**Required:** Yes  

### Rules
- Must not be empty.
- Allows letters and numbers.
- Allows common punctuation (`. , - # /`).
- Minimum **5 characters**.
- Maximum **150 characters**.

### Example Regex
^[A-Za-z0-9\s.,#-]{5,150}$


---

## 6. Age 🔴

**Required:** Yes  

### Rules
- Must not be empty.
- Must be a whole number.
- No decimal values allowed.
- Allowed range: **0–120**.

### Example Regex
^(?:1[01][0-9]|120|[1-9]?[0-9])$


---

## 7. Gender 🔴

**Required:** Yes  

### Rules
- Only one option can be selected.
- Allowed values:
  - `Male`
  - `Female`
